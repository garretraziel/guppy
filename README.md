Konvence
--------

### Odsazování
Každá úrověň 4 mezery, žádné taby.

### Styl psaní bloků
    int main(void)
    {
        if() {
            ;
        } else {
            ;
        }
        return 0;
    }
Funkce mají blok až na dalším řádku. Cykly a podmíněné příkazy hned. Else takhle vypadá prďácky.

### Ostatní
Řádky max 80 znaků, to známe.

Konce radku unixove (lf)
